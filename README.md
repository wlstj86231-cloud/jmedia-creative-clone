# J MEDIA Creative Clone

J MEDIA 브랜드로 새로 만든 크리에이티브모어 레퍼런스형 정적 사이트입니다.

## 1차 구현 범위

- 고정 헤더와 모바일 메뉴
- 풀뷰포트 히어로
- 소개 섹션
- 하이라이트 필터형 포트폴리오
- J MEDIA WORLD 인터랙션 섹션
- 운영 사이트 마키 섹션
- About / Approach / Contact 페이지
- EmailJS 기반 상담 신청 폼
- 정보형 / 도구형 / 지도형 / 참여형 하이라이트 상세 페이지

## 기준

- 원본 사이트의 코드, 문구, 이미지를 그대로 복제하지 않습니다.
- 섹션 흐름, 큰 인터랙션 감각, 타이포그래피 밀도만 참고합니다.
- J MEDIA의 포트폴리오 이미지와 자체 카피로 구성합니다.

## 상담 폼

`contact.html`에서 J MEDIA 전용 EmailJS 설정을 사용합니다. 기존 policyfundpedia.com 파일이나 템플릿 파일은 수정하지 않습니다.
